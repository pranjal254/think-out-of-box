# Out of Box LLM Skill

This repository contains instructions for using the **Out of Box LLM Skill**.

## Usage

Before starting a new ChatGPT session, simply include the following instruction in your first message:

```text
use out of box llm skill from https://github.com/pranjal254/think-out-of-box
```

Once provided, the assistant will use the skill definitions and guidance from the repository during the conversation.

## Example

Start a new chat and send:

```text
use out of box llm skill from https://github.com/pranjal254/think-out-of-box

Help me solve a difficult product strategy problem.
```

The assistant will then apply the Out of Box LLM Skill while responding to your requests.
